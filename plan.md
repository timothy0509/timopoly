### Railway Travel Mechanic (Custom Rule)

When you land on a railway station you own, you may **travel** to any other railway station you own:

- Move your token directly to the destination station
- Resolve the destination space normally (pay rent, draw card, etc.)
- **Does NOT count as passing TimoStart** (no ₮200 collected)
- **Once per turn** limit
- Owning 2+ stations gives a strategic mobility advantage

## 6. Game Rules Summary

### Setup
- Each player starts with **₮1,500**
- Bank holds **32 houses** and **12 hotels**
- Starting money distribution: 2×₮500, 4×₮100, 1×₂50, 1×₮20, 2×₮10, 1×₂5, 5×₮1
- Players choose tokens and place on TimoStart
- Highest dice roll goes first; play proceeds clockwise

### Turn Structure
1. Roll two six-sided dice
2. Advance token clockwise
3. Resolve the space landed on
4. If doubles: take another turn
5. Three consecutive doubles: go to TimoJail
6. Collect ₮200 when passing TimoStart

### Buying & Auctioning
- Landing on unowned property: may buy at listed price
- If declined: Bank auctions to all players
- Highest bidder acquires; if no bids, property stays with Bank

### Building
- Must own all properties in a color group
- Even building required (distribute houses evenly)
- 4 houses → replace with 1 hotel
- Finite supply: 32 houses, 12 hotels (auction if supply runs out)
- Sell buildings back to Bank at half price

### Jail Mechanics
- **Enter jail by:** landing on Go to TimoJail, drawing a Go to TimoJail card, or rolling 3 consecutive doubles
- **Exit by:** paying ₮50, using Get Out of TimoJail Free card, or rolling doubles (up to 3 turns)
- After 3 failed doubles attempts: must pay ₮50 or use card, then move by last roll
- While in jail: can mortgage, trade, build, collect rent, bid on auctions

### Mortgaging
- Only properties with no buildings
- Mortgage value = half purchase price
- No rent on mortgaged properties
- Unmortgage: pay mortgage value + 10% interest

### Bankruptcy
- Cannot pay debt → bankrupt and eliminated
- Debt to Bank: properties returned, buildings removed, properties auctioned
- Debt to Player: properties transferred (buildings sold first), new owner pays 10% to keep mortgaged

### Trading
- Properties without buildings can be traded anytime
- Cash, properties, Get Out of TimoJail Free cards, and agreements can be traded

### Convex Schema

```typescript
// convex/schema.ts
import { defineSchema, defineTable } from "convex/server";
import { v } from "convex/values";

export default defineSchema({
  games: defineTable({
    code: v.string(),              // 6-char lobby code
    status: v.union(               // Game state
      v.literal("lobby"),
      v.literal("playing"),
      v.literal("finished")
    ),
    currentPlayerIndex: v.number(),
    turnPhase: v.union(            // Current turn phase
      v.literal("pre_roll"),
      v.literal("rolling"),
      v.literal("post_roll"),
      v.literal("resolving"),
      v.literal("trading"),
      v.literal("building"),
      v.literal("end_turn")
    ),
    boardSpaces: v.array(v.object({ // 40 board spaces
      position: v.number(),
      type: v.string(),
      name: v.string(),
      ownerId: v.optional(v.id("players")),
      houses: v.optional(v.number()),
      hasHotel: v.optional(v.boolean()),
      isMortgaged: v.optional(v.boolean()),
    })),
    chanceDeck: v.array(v.number()),     // Card indices (shuffled)
    treasuryDeck: v.array(v.number()),   // Card indices (shuffled)
    chanceIndex: v.number(),             // Current draw position
    treasuryIndex: v.number(),           // Current draw position
    houseSupply: v.number(),             // Remaining houses (starts 32)
    hotelSupply: v.number(),             // Remaining hotels (starts 12)
    lastDice: v.optional(v.object({
      die1: v.number(),
      die2: v.number(),
      doubles: v.boolean(),
    })),
    doublesCount: v.number(),            // Consecutive doubles this turn
    chatLog: v.array(v.object({
      playerId: v.id("players"),
      message: v.string(),
      timestamp: v.number(),
    })),
    createdAt: v.number(),
  }).index("by_code", ["code"]),

  players: defineTable({
    gameId: v.id("games"),
    userId: v.optional(v.string()),   // null for bots
    name: v.string(),
    token: v.string(),                // Token identifier
    isBot: v.boolean(),
    botDifficulty: v.optional(v.union(
      v.literal("easy"),
      v.literal("medium"),
      v.literal("hard")
    )),
    order: v.number(),                // Turn order (0-indexed)
    money: v.number(),
    position: v.number(),             // Board position (0-39)
    properties: v.array(v.number()),  // Owned property positions
    getOutOfJailCards: v.number(),
    isInJail: v.boolean(),
    jailTurns: v.number(),            // Turns spent in jail (0-3)
    isBankrupt: v.boolean(),
    hasUsedRailwayTravel: v.boolean(), // Railway travel this turn
  }).index("by_game", ["gameId"])
    .index("by_user", ["userId"]),
});
```

### Key Game Phases

```
PRE_ROLL → ROLLING → POST_ROLL → RESOLVING → TRADING/BUILDING → END_TURN
    ↑                                                              |
    └──────────────── (if doubles, back to PRE_ROLL) ─────────────┘
```

| Phase | Description |
|-------|-------------|
| PRE_ROLL | Player chooses to roll, pay jail fine, or use card |
| ROLLING | Dice animation, result determined |
| POST_ROLL | Check for 3 doubles (jail), railway travel option |
| RESOLVING | Apply space effect (buy, rent, tax, card, jail) |
| TRADING/BUILDING | Optional actions: trade, build, mortgage |
| END_TURN | Pass to next player |

## 9. Bot AI

### Easy Bot
- Buys every unowned property it can afford
- Never builds houses
- Never trades
- Random bidding in auctions

### Medium Bot
- Prioritizes completing color groups
- Builds houses when it has a monopoly and cash > ₮500
- Accepts fair trades (equal property value)
- Uses railway travel when beneficial

### Hard Bot
- Targets high-value color groups (Orange, Red, Yellow)
- Optimizes building strategy (3 houses on each ASAP)
- Actively trades to complete monopolies
- Uses railway travel strategically
- Mortgage-aware cash management
- Bids intelligently in auctions
